# Hangman-Forca-PTBR-
Hangman game written in C for the terminal, featuring 250 words divided into 5 randomly selected themes. The player tries to guess the word letter by letter with up to 6 attempts, watching the hangman being drawn with each wrong answer. Colored interface using ANSI Escape Codes.

🇧🇷 Português
Descrição
Jogo da Forca desenvolvido em linguagem C para terminal, com interface colorida usando ANSI Escape Codes. O jogador tenta adivinhar palavras sorteadas aleatoriamente, com dicas de tema para auxiliar.
Funcionalidades

250 palavras divididas em 5 temas
Tema sorteado exibido como dica na tela
Boneco desenhado progressivamente a cada erro
6 tentativas antes do enforcamento
Interface colorida no terminal

Como compilar e executar
bashgcc main.c -o forca
./forca
Requisitos

Compilador GCC
Terminal com suporte a ANSI Escape Codes
